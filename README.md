## 퍼플이
KAIST Puple 서버의 봇입니다.

### 기능   
- `퍼플이 안녕`: 퍼플이에게 인사를 합니다.
- `!식당`: 식당 중 하나를 무작위로 선택합니다.  
    - `!식당 [장소]`: 입력받은 장소 내의 식당 중 하나를 무작위로 선택합니다. 디폴트는 대전 어은동입니다.
